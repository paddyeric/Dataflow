Redux Practice Project

This is a React application built to practice Redux Toolkit for state management. It simulates a simple blog or social media platform where users can add posts, view them in a list, and see details like the author and time since posting.

Features

- Add Posts: Users can create new posts with a title, content, and select an author from a predefined list.
- Display Posts: Posts are displayed in a sorted list (newest first), showing the title, a preview of the content, author name, and time ago since creation.
- State Management: Uses Redux Toolkit for managing posts and users state.
- Components: Modular components for adding posts, listing posts, displaying author, and formatting timestamps.

Technologies Used

- React 19
- Redux Toolkit
- date-fns for date formatting
- Create React App for project setup
